Javascript Cheatsheet

Objects

var person = {firstName:"Jarred", lastName:"Hare", age:25, eyeColor:"green"};
var person = new Object();
person.firstName = "Jarred";
person.lastName = "Hare";
person.age = 25;
person.eyeColor = "green";

An object in JS is essentially a container that has properties and methods associated with it. In the example here the properties of the "person" object are set using object literal notation or the constructor function.

We can easily add and refer to properties and methods to the object using the . (dot) notation. In the example we have seen we use this notation to create the firstName property on the object person.

Prototypes

var Person = function Person(firstName, lastName, age, eyeColor) {
  this.firstName = firstName;
  this.lastName = lastName;
  this.age = age;
  this.eyeColor = eyeColor;
}

Person.prototype.doubleAge = function() { return this.age * 2 }

Every function is assigned a new empty object called prototype. This object has some special treatment, the key point is that when any new object is created it inherits the properties of its constructor's prototype. Like any other object, we can assign both properties and methods to the prototype. Properties are usually defined in the constructor method whereas methods are normally defined using the prototype object.
